Bank Loan Report

This project analyzed bank loan data using SQL to derive key performance indicators (KPIs) and visualized the findings in a Tableau dashboard.

The analysis addressed two main areas: loan portfolio performance and loan status analysis. KPIs included total loan applications, funded amounts, amounts received, average interest rates, and debt-to-income ratios, with month-to-date (MTD) and month-over-month (MoM) comparisons.

The analysis further segmented loans into "Good" (Fully Paid, Current) and "Bad" (Charged Off) categories, calculating relevant metrics for each.

Housing Data Cleaning

This project involved cleaning and structuring messy housing data from Nashville, TN, into an analysis-ready format using SQL.

Key tasks included standardizing date formats, addressing missing values with relevant data or placeholders, and performing data type conversions. Common Table Expressions (CTEs), window functions, joins, and subqueries were utilized to streamline queries and combine data from multiple sources.

These techniques ensured improved accuracy, consistency, and usability of the dataset. The final product is a high-quality dataset optimized for analysis, highlighting SQL expertise and meticulous attention to detail. This project demonstrates essential data cleaning skills critical for analytics and decision-making.

Amazon Web Scrapping

The Amazon Web Scraper Project is a Python-based tool designed to extract and monitor product information from Amazon.

Utilizing libraries such as BeautifulSoup and Requests, the scraper collects data including product titles, prices, and availability. This information is stored in a CSV file and is periodically updated to reflect the latest details.

The project aims to automate the process of tracking product prices over time, providing users with valuable insights into price fluctuations and assisting in making informed purchasing decisions.
