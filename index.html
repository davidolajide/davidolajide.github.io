Housing Data Cleaning

This project involved cleaning and structuring messy housing data from Nashville, TN, into an analysis-ready format using SQL.

Key tasks included standardizing date formats, addressing missing values with relevant data or placeholders, and performing data type conversions. Common Table Expressions (CTEs), window functions, joins, and subqueries were utilized to streamline queries and combine data from multiple sources.

These techniques ensured improved accuracy, consistency, and usability of the dataset. The final product is a high-quality dataset optimized for analysis, highlighting SQL expertise and meticulous attention to detail. This project demonstrates essential data cleaning skills critical for analytics and decision-making.
